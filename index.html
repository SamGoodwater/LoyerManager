          Avant de commencer

          Loyer Manager — sur votre serveur, sans cloud.

          Loyer Manager vous aide à suivre les loyers, enregistrer les paiements et produire des quittances. Vos données restent sur votre serveur — pas de cloud tiers.

          Accès : connectez-vous avec votre compte, puis renseignez vos paramètres.
          Import rapide : glissez-déposez un fichier .json (profil) ou .csv (relevé bancaire) n'importe où sur la page.

          Étape 1 Premiers pas

          Connectez-vous, puis ouvrez Paramètres pour renseigner bailleur, locataire et montant du loyer.

          1. Créez votre compte (passphrase) ou connectez-vous avec Google ou Microsoft à la première visite.
          2. Remplissez les sections Paramètres (bailleur, locataire, loyer).
          3. Cliquez sur Enregistrer en bas à droite pour enregistrer tout de suite (sinon, sauvegarde automatique sous 30 s).

          Étape 2 Paramétrer l'application

          Dans l'onglet Paramètres, remplissez les sections ci-dessous. Vos modifications sont enregistrées automatiquement ; utilisez Enregistrer en bas à droite pour forcer la sauvegarde.

          2.1 — Location

          • Début de location : date de prise d'effet du bail.
          • Jour habituel de paiement : jour où le locataire règle le loyer (souvent le 1er du mois).

          2.2 — Bailleur et locataire

          Renseignez les noms et adresses : ils apparaîtront sur la quittance de loyer.

          2.3 — Signature

          Importez une photo ou un scan de votre signature (PNG/JPG, max. 5 Mo). Elle sera placée en bas de la quittance.

          2.4 — Historique des loyers

          Pour chaque palier, renseignez le loyer hors charges, les charges locatives et la date d'application. Le total mensuel alimente le tableau de bord et les quittances, avec le détail loyer + charges (conformément aux obligations du bailleur — voir Service Public — quittance de loyer).

          2.5 — Émetteurs (pour l'import bancaire)

          Si vous importez un relevé CSV, configurez le nom du locataire et les mots à repérer dans le libellé bancaire (ex. Jean Dupont).

          2.6 — Modèles quittance et mail

          Dans Paramètres → Modèles, gérez vos modèles. Deux modèles de base sont toujours présents : complet (détaillé, défaut) et court (simplifié). Ils sont en lecture seule : aperçu et export uniquement. Pour personnaliser, utilisez + Nouveau modèle ou Importer modèle (crée un nouveau modèle, ne remplace jamais les modèles de base). L'édition du contenu se fait dans les onglets Quittance et Mail (mode « Édition du modèle »). Le panneau Mots-clés propose les mêmes variables pour quittance et mail ({{paiement}}, {{bailleur.name}}, etc.).

          2.7 — Destinataires mail

          Indiquez les adresses e-mail (À, CC, CCI) et la signature texte ({{signature}}).

          2.8 — Envoi mail (Gmail / Outlook)

          Connectez votre compte pour envoyer directement ou enregistrer un brouillon (relecture dans votre messagerie avant envoi). Reconnectez le compte si le bouton Brouillon reste inactif après une mise à jour.

          2.9 — Envoi mail (SMTP)

          Alternative sans OAuth pour l'envoi direct uniquement. Les brouillons dans la boîte mail passent par le menu Autres → EML + PDF.

          Étape 3 Enregistrer les paiements

          Deux méthodes — choisissez celle qui vous convient.

          Méthode A — Import depuis votre banque (CSV)

          1. Connectez-vous à votre banque en ligne et exportez vos opérations au format CSV.
          2. Dans l'onglet Paiements, cliquez sur Importer CSV et choisissez le fichier.
          3. Cochez les lignes à enregistrer (les doublons sont déjà exclus).
          4. Cliquez sur Importer la sélection.
          L'application repère les paiements du locataire grâce aux noms configurés dans Paramètres (section « Émetteurs pour l'import bancaire »).

          Méthode B — Saisie manuelle

          1. Cliquez sur Ajouter un paiement.
          2. Indiquez la date, le montant, le type (virement, espèce…) et l'émetteur.
          3. Cliquez sur Enregistrer.

          La saisie manuelle convient pour corriger une ligne ou enregistrer un paiement en espèces, sans fichier bancaire.

          Étape 4 Consulter les données

          L'onglet Tableau de bord centralise la vue d'ensemble.

          • Utilisez la barre de période sous le menu : mois, bouton « Période » pour une plage Du→Au, timeline colorée.
          • Les indicateurs résument le solde à date, le recouvrement et les mois problématiques.
          • Le tableau « Détail par mois » affiche pour chaque mois ce qui était dû, reçu, l'écart et le solde. L'icône crayon ouvre le détail du mois.
          • Le calendrier coloré reprend les mêmes statuts mois par mois — cliquez une case pour sélectionner ce mois.
          • Les graphiques et le récapitulatif annuel reprennent les mêmes chiffres sous forme visuelle.
          • Les icônes imprimer en haut à droite de chaque bloc ouvrent une impression ciblée. Le rapport Tableau de bord regroupe indicateurs, détail par mois et paiements du mois. Imprimer le rapport complet ajoute le graphique et le récapitulatif annuel.

          Étape 5 Quittance et export

          1. Choisissez le mois dans l'onglet Quittance (ou Tableau de bord / Mail — même sélecteur).
          2. L'aperçu se remplit automatiquement à partir du modèle choisi.
          3. Pour modifier le modèle : basculez en Édition du modèle (modèles personnalisés uniquement — complet et court sont en lecture seule).
          4. Exportez :
            • PDF — le plus courant pour envoyer ou imprimer ;
            • DOCX — pour modifier dans Word ;
            • HTML — page web autonome ;
            • Exporter plusieurs… — un seul fichier PDF, DOCX ou HTML pour une plage de mois.
          Cliquez sur Régénérer après avoir modifié des paiements ou des paramètres pour recalculer l'aperçu.

          Étape 6 Mail et envoi

          1. Configurez les destinataires dans Paramètres → Destinataires et signature mail.
          2. Connectez Gmail/Outlook (Paramètres → Envoi mail) ou renseignez le SMTP pour l'envoi direct.
          3. Choisissez le mois dans l'onglet Mail (barre de période partagée).
          4. Personnalisez le modèle si besoin : Nouveau modèle…, Importer modèle, mode Édition du modèle.
          5. Brouillon — mail + PDF dans Gmail/Outlook (OAuth). Envoyer — envoi immédiat. Autres — EML + PDF ou mailto.
          Avec le SMTP seul, seul l'envoi direct est possible ; utilisez EML + PDF pour préparer un brouillon local.

          Étape 7 Historique

          L'onglet Historique retrace les actions importantes : mails envoyés, brouillons créés, imports CSV, exports PDF/DOCX.

          • Filtrez par type pour retrouver une action.
          • Exporter CSV — archive pour vos dossiers ou la conformité RGPD.
          • Purger tout — efface le journal (irréversible).
          • Ajustez la conservation (défaut 24 mois ; 0 = illimitée).

          Confidentialité (RGPD)

          Loyer Manager est auto-hébergé : vous êtes responsable du traitement des données sur votre instance.

          Données stockées

          • Vos loyers, paiements et paramètres (exportables depuis Mon compte).
          • Le journal d'activité (onglet Historique).
          • Vos modèles de quittance et de mail.

          Cookies et stockage navigateur

          • Session de connexion — uniquement pendant que vous utilisez l'application.
          • Mémorisation de votre choix sur le bandeau de confidentialité (pas de publicité).

          Vos droits

          Export du profil (Paramètres → Mon compte), export CSV et purge depuis l'onglet Historique. Déconnexion du compte mail dans Paramètres → Envoi mail. Conservation par défaut : 24 mois (modifiable dans Historique).

          Installation sur un serveur

          Loyer Manager s'installe sur un hébergement web avec PHP. Vos données restent sur votre serveur.

          Première utilisation

          1. Ouvrez l'application via son adresse web.
          2. Créez votre compte (passphrase) ou connectez-vous avec Google ou Microsoft.
          3. Changez la passphrase dans Paramètres → Mon compte (compte local).
          4. Renseignez vos paramètres dans l'onglet dédié.

          En production, activez HTTPS et protégez l'accès au site (mot de passe hébergeur recommandé).

          Dépannage

          Les modèles ne s'enregistrent pas

          Vérifiez votre connexion et réessayez. Contactez l'administrateur du serveur si le problème persiste.

          Impossible de modifier un modèle de base

          C'est normal : les modèles complet et court sont en lecture seule. Dupliquez-les ou importez un fichier pour créer une variante personnalisée.

          Je ne peux plus me connecter

          Vérifiez votre passphrase ou reconnectez-vous avec Google ou Microsoft. En cas d'oubli sur un compte local, l'administrateur peut supprimer le compte côté serveur pour permettre une nouvelle inscription.

          Mes données semblent endommagées

          Au démarrage, l'application peut vous proposer de télécharger une copie de secours puis de repartir sur une base vide.

          L'import bancaire ne trouve aucun paiement

          Vérifiez la section « Émetteurs pour l'import bancaire » dans Paramètres : les mots-clés doivent correspondre au libellé affiché par votre banque.

          Les chiffres du tableau de bord semblent incorrects

          Vérifiez la date de début de location, l'historique des loyers et que tous les paiements ont bien été enregistrés.

          Le bouton Brouillon est grisé

          Les brouillons dans la boîte mail nécessitent Gmail ou Outlook (OAuth). Avec le SMTP seul, utilisez Autres → EML + PDF. Si OAuth est connecté mais le bouton reste inactif, déconnectez puis reconnectez le compte mail.

          L'envoi mail échoue

          Vérifiez les destinataires dans Paramètres, la connexion Gmail/Outlook ou la configuration SMTP. Consultez l'onglet Historique pour le détail de l'erreur.

          Exporter mon profil

          Dans Paramètres → Sauvegarde JSON, exportez ou importez votre fichier de données (paramètres et paiements). Pour une sauvegarde complète protégée par mot de passe, utilisez Mon compte.

          Exporter plusieurs quittances

          Génère un fichier unique contenant une quittance par mois dans la plage choisie (modèle sélectionné dans l'onglet).
